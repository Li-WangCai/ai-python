- miniconda 创建一个叫mathy的环境
  ```bash
  conda create -n mathy --file mathy.yaml
  ```
- 进入mathy并且安装相应的包 
  ```bash
  conda activate mathy
  conda install python12
  ```
- 每次运行jupyter-lab都执行以下命令更新
  ```bash
  cd; conda deactivate; sudo apt update; sudo apt upgrade; \
  conda update -n base -c defaults conda; conda activate mathy; \
  conda update --all; jupyter-lab
  ```
